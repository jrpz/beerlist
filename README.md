# The Beer List

The Beer List retrieves and displays [Untappd](https://untappd.com) user information. To start things off, I've supplied my UserName by default.

#### Tools Used
* [Untappd API](https://untappd.com/api/docs/v4)
* [Yeoman](http://yeoman.io/)
* [Bower](http://bower.io)
* [Gulp](http://gulpjs.com)
* [Gulp Web app generator](https://github.com/yeoman/generator-gulp-webapp)
* [Backbone.js](http://backbonejs.org/)
* [Underscore.js](http://underscorejs.org/)
